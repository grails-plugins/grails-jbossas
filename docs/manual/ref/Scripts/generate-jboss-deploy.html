generate-jboss-deploy

Purpose

Generate deployment descriptor(s) for the specified version of JBoss.

Description

Usage: 

grails generate-jboss-deploy <version>

Examples:

Generate WEB-INF/jboss-web.xml for JBoss 5:

grails generate-jboss-deploy 5

Generate WEB-INF/jboss-classloading.xml and WEB-INF/jboss-scanning.xml for JBoss 6:

grails generate-jboss-deploy 6

Required arguments:

  • version - The major version of JBoss, either 5 or 6

Optional arguments: none .
